使用须知：
本游戏所有mod均需依赖Melon Loader以运行

对mod设置的修改，在游戏目录下UserData文件夹中MelonPreferences.cfg文件进行

物品权重编辑：
允许玩家设置不同品质物品的生成概率占比

    仅可控制武器与模块等具有稀有度的物品生成

    mod改变的是生成权重而非概率，其算法为指定品质稀有度权重/(Common权重+Rare权重+Epic权重+Legendary权重)
    
    mod无法控制生成几率，即资源点是否生成物品，生成物品的数量

    mod无法控制不同类型物品的生成权重，即一个资源点是生成消耗品还是有稀有度的物品

    mod无法调整同一稀有度下不同物品各自的生成权重，这是由于物品各自权重被写在了另一个文件中

    物品自带的词条权重与物品本身的权重是两个分开的设置项，别忘了一起修改

    由于官方近期更新改动了物品生成机制，所以将部分稀有度权重设为0会低概率导致箱子中不生成任何物品

物品模块编辑：
可以修改武器与组件模块，甚至凭空生成物品

    该mod尚不完善，存在诸多问题。包括但不限于：

    编辑模块时允许选择不属于该物品的模块

    编辑模块时缺少部分可用于该物品的模块

    生成物品时列表不全

    生成武器时会出现瞄具丢失bug（指退出重进游戏后武器瞄具消失）

化学弹头bug修复：
原版化学弹头模块升级会提高触发阈值同时降低触发后的伤害，mod修复了这一问题

    有传闻，该mod会导致其它模块失效，暂未核实该问题
