# Jump Space Item Editor v3.1 - 指令参考

## 命令格式

所有命令以 `ie` 开头，格式：`ie <子命令> [参数...]`

---

## 一、武器编辑指令

需要先手持武器并执行 `ie load` 加载。

### ie load
加载当前手持武器，显示武器摘要信息（名称、类型、稀有度、等级、模块数量）。

快捷键：**F5**

### ie reload
重新加载手持武器（等同于 `ie load`）。

### ie info
显示当前武器的详细信息，包括每个模块的名称、稀有度、升级等级。

### ie base
列出所有基础模块槽的选项摘要（当前模块 + 可选数量）。

### ie base \<槽号\>
列出指定基础模块槽的详细选项（分页显示，每页20个）。
- 槽号从1开始
- 枪械有2个基础模块槽，近战有1个

### ie base \<槽号\> \<编号\> [等级] [品质]
选择基础模块替换当前槽位。
- **编号**：选项列表中的序号（从1开始）
- **等级**：可选，指定模块升级等级（默认沿用当前等级）
- **品质**：可选，指定模块品质（0=Common, 1=Rare, 2=Epic, 3=Legendary，默认使用模块最高可用品质）

示例：
```
ie base 1 5        # 基础模块槽1替换为选项5，沿用当前等级和最高品质
ie base 1 5 3      # 基础模块槽1替换为选项5，等级3
ie base 1 5 3 2    # 基础模块槽1替换为选项5，等级3，Epic品质
```

### ie addon
列出所有附加模块槽的选项摘要。

### ie addon \<槽号\>
列出指定附加模块槽的详细选项。

### ie addon \<槽号\> \<编号\> [等级] [品质]
选择附加模块替换当前槽位。参数含义同 `ie base`。

### ie rarity [0-3]
设置目标稀有度，将在 `ie apply` 时应用到武器。
- 不带参数：查看当前稀有度和待应用稀有度
- **0** = Common（普通）
- **1** = Rare（稀有）
- **2** = Epic（史诗）
- **3** = Legendary（卓越）

### ie apply
应用所有待应用的修改（模块替换 + 稀有度变更）。

### ie pending
查看当前所有待应用的修改列表。

### ie reset
取消所有待应用的修改，恢复到加载时的状态。

### ie modules [base|addon]
搜索/浏览模块池。
- 无参数：显示所有模块（基础+附加，最多50个）
- **base** / **b**：仅显示基础模块
- **addon** / **a** / **add**：仅显示附加模块

## 二、物品生成指令（层级式浏览）

### ie spawn
进入物品生成浏览模式（等同于 `ie spawn list`）。

### ie spawn list
返回顶级分类列表，显示：武器、组件、神器、自定义分组。

### ie spawn \<编号\>
根据当前浏览层级，选择对应项目：
- **顶级**：进入对应类别（1=武器, 2=组件, 3=神器, 4=自定义分组）
- **武器子分类**：进入对应武器类型的列表
- **武器列表**：选择武器 → 进入瞄具选择
- **组件子分类**：进入对应组件类型的列表
- **组件列表**：直接生成对应组件
- **神器列表**：直接生成对应神器
- **自定义分组列表**：进入对应分组的物品列表
- **分组物品列表**：自动识别类型并生成对应物品

### ie spawn \<类别名称\>
在顶级或子分类层级，按名称进入类别：
- 顶级：`武器`/`weapon`/`w`、`组件`/`component`/`comp`/`c`、`神器`/`artifact`/`a`、`自定义分组`/`custom`/`cg`
- 武器子分类：输入子分类名称
- 组件子分类：输入子分类名称
- 自定义分组列表：输入分组名称

### ie spawn back
返回上一级浏览层级。

### ie spawn guid \<GUID\>
按GUID直接生成武器/神器。
- 自动尝试从目录和DevEquipper查找
- 使用当前设置的生成品质

### ie spawn comp \<GUID\>
按GUID直接生成飞船组件。
- 自动尝试从目录和DevEquipper查找
- 使用当前设置的生成品质

### ie spawn rarity [0-3|max|common|rare|epic|legendary]
设置/查看生成品质。
- 不带参数：查看当前品质设置
- **max** / **m**：设为最高允许品质（默认，自动使用物品允许的最高品质）
- **0-3**：设为指定品质
- **common/rare/epic/legendary**：按名称设置品质

### 瞄具选择
选择武器后自动进入瞄具选择界面：
- **0**：随机瞄具（默认）
- **1-N**：选择指定瞄具
- 输入 `ie spawn back` 返回武器列表

## 三、翻译与本地化指令

### ie reload_trans
重新加载翻译文件。修改翻译文件后执行此命令刷新。

### ie import_trans \<文件名\>
从Mods目录导入额外的翻译文件。
- 文件格式：每行一条，`英文原名=中文翻译`
- 示例：`ie import_trans my_weapons.txt`

### ie trans_status
查看翻译状态，包括：
- 已加载翻译条数
- 翻译文件是否存在
- 已加载的翻译文件列表
- 当前武器的翻译覆盖率

翻译文件位置：
- 主文件：`Mods/module_translations.txt`
- 自动扫描：`Mods/module_translations_*.txt`

## 四、模块池配置指令

### ie reload_modules
重新加载模块池配置文件。

### ie module_pools
查看模块池配置状态，显示各类别模块数量和加载来源。

配置文件路径（Mods/目录下）：
| 文件名 | 类别 |
|--------|------|
| infantry_weapon_modules.txt | 步战武器模块 |
| ship_weapon_modules.txt | 舰载武器模块 |
| engine_modules.txt | 引擎模块 |
| shield_modules.txt | 护盾模块 |
| power_modules.txt | 电力模块 |
| materia_modules.txt | Materia模块 |
| turret_system_modules.txt | 炮塔系统模块 |
| reactor_modules.txt | 反应炉模块 |

## 五、自定义生成分组指令

### ie spawn_groups
查看自定义生成分组列表，显示每个分组名称和物品数量。

### ie reload_spawn_groups
热重载自定义生成分组文件，扫描 `Mods/spawn_groups_*.txt`。

### 生成分组文件格式

文件位置：`Mods/spawn_groups_<名称>.txt`

每个文件代表一个分组，文件名（去掉前缀和扩展名）作为分组显示名。

```
# 注释行（#开头）
# 每行格式: GUID=显示名称
abc123def456=AK-47
def456abc789=M4A1

# 也可以只写GUID（显示名称留空，生成时从目录查找）
xyz789abc456
```

### 生成默认值
- **品质**：Legendary（受ExcludedRarities钳制）
- **物品等级**：1
- **模块升级等级**：1

## 六、浏览层级结构

```
顶级 (ie spawn list)
├── 武器
│   ├── 子分类（突击步枪/霰弹枪/狙击枪/...）
│   │   └── 武器列表
│   │       └── 瞄具选择 → 生成
│   └── ...
├── 组件
│   ├── 子分类（引擎/护盾/电力/...）
│   │   └── 组件列表 → 生成
│   └── ...
├── 神器
│   └── 神器列表 → 生成
└── 自定义分组
    ├── 分组列表
    │   └── 分组物品列表 → 自动识别类型并生成
    └── ...
```

## 八、快捷操作

| 操作 | 命令 |
|------|------|
| 加载手持武器 | F5 或 `ie load` |
| 查看帮助 | `ie help` |
| 按GUID生成武器 | `ie spawn guid <GUID>` |
| 按GUID生成组件 | `ie spawn comp <GUID>` |
| 设置生成品质为Legendary | `ie spawn rarity 3` |
| 恢复自动品质 | `ie spawn rarity max` |
| 返回顶级浏览 | `ie spawn list` |
| 返回上一级 | `ie spawn back` |
